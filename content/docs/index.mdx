## What is POKÉ LOCALS?

POKÉ LOCALS is a web-based platform designed to streamline the management and organization of Pokémon tournaments. It provides tools for:

- Tournament organization and management
- Tournament registration (comming soon)
- Tournament pairings
- Discord integration for announcements
- QR code generation for easy access
- Real-time tournament updates

## Beta Notice

POKÉ LOCALS is currently in beta. We're continuously improving the platform based on user feedback and adding new features regularly.
