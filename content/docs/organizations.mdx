description: Organizations in POKÉ LOCALS represent tournament organizer groups or local game stores that run Pokémon tournaments. Each organization can have multiple members with different roles and permissions.

## Creating an Organization

To create a new organization:

1. Click on the "Organization" tab in your profile
2. If you don't have an organization, you'll see a form to create one
3. Fill in the required information:
   - Organization name (Required)
   - Website (Required)
   - Location (Optional)
   - Average tournament size (Required)
   - Message (Optional)
4. Click "Submit" to create your organization

## Managing Organization Members

### Adding Members

1. Navigate to your organization's profile
2. Find the "Organization members" section
3. Enter the user's email in the "User email" field
4. Click "Save"

### Member Roles

Members can have one of two roles:

- **Admin**: Can manage organization settings, add/remove members, and create tournaments
- **Member**: Can create and manage tournaments

### Removing Members

To remove a member:

1. Find the member in the member list
2. Click the "Remove" button next to their name

## Discord Integration

Organizations can set up Discord webhooks to automatically notify Discord servers about tournaments.

### Setting Up Discord Notifications

1. Navigate to the "Discord notifications" section
2. Click "Add webhook"
3. Enter:
   - Webhook Name - will be used while sending notifications
   - Discord Webhook URL (ask administrator of your Discord server to provide an url)
4. Click "Save"

Multiple webhooks can be configured to notify different Discord servers.

## Organization Selector

The organization selector in the top header allows you to:

- Switch between different organizations you're a member of
- Access organization-specific features and tournaments
- View your current organization context

## FAQ

### Can I be part of multiple organizations?

Yes, you can be a member of multiple organizations and switch between them using the organization selector.

### How do I transfer organization ownership?

Currently, organization ownership transfer must be handled by POKÉ LOCALS support.

### Can I change my organization name?

Currently, organization name change must be handled by POKÉ LOCALS support.
